Aprendendo CSS

Pode se adicionar estilos com css das seguintes formas:

  • Atributo dentro da tag: , por exemplo: < p style="color:blue">
  • Tag na head:

    • tag style

  • Arquivo separado, com link na head: "< link href="./css/main.css" ref"stylesheet">"

Outras características (ver no código):

  • Classes css: ".minha_classe"
  • Id css: "#meu_id"
